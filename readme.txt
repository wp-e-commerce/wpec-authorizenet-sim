== Description ==

Authorize.net Server Integration Method (SIM) provides a secure hosted
payment form to make integration easy for Web merchants that do not
have an SSL certificate.  The Authorize.Net Payment Gateway can handle
all the steps in the secure transaction process:

	* Secure payment data collection
	* Secure transmission of payment information for settlement
	* Secure cardholder information storage
